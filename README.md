# CSP451 CheckPoint 1 - Version Control and Github Basics

This project is part of CheckPoint 1 for CSP451. It demonstrates the fundamentals of Git, GitHub, version control workflows, branching pull requests, and documentation using Markdown

## Project Structure

- 'index.html' - Main webpage
- 'about.html' - Added later in feature branch
- 'style.css' - Basic styling
- 'script.js' - JAvaScript console output
- '.gitignore' -Ignore patterns
- 'README.md' - Project documentation
